# HideArmour
超轻量级 mod，在客户端侧隐藏自己的护甲，并且可以指定4个护甲槽分别是否需要隐藏

旧的MOD出现了许多问题，所以被我抛弃了，直接重写

# 使用方式
将本 mod 放入 `mods` 目录下，启动游戏

## 1.8.9至1.12.2版本
使用客户端命令`/hidearmour`或`/hidearmor`来操作

- `true` - 启用隐藏护甲
- `false` - 禁用隐藏护甲
- `status` - 查看本MOD的状态
- `reload` - 重载

配置文件位于`.minecraft/config/hidearmour.cfg`
